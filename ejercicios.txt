Ejercicio 1: Calculadora de IMC (Índice de Masa Corporal)
Crea un programa que calcule el Índice de Masa Corporal (IMC) de una persona. El IMC se calcula con la fórmula:

IMC= peso /altura 2
IMC= altura 2 peso
​
 
Donde el peso está en kilogramos y la altura en metros.

Pide al usuario que ingrese su peso y altura.

Calcula el IMC.

Muestra un mensaje indicando si la persona tiene:

Bajo peso (IMC < 18.5)

Peso normal (18.5 ≤ IMC < 25)

Sobrepeso (25 ≤ IMC < 30)

Obesidad (IMC ≥ 30)

--------------------------------------------

Ejercicio 2: Conversor de Monedas
Crea un programa que convierta una cantidad de dinero de una moneda a otra. El programa debe:

Preguntar al usuario la cantidad de dinero, la moneda de origen y la moneda a la que desea convertir.

Utilizar tasas de cambio fijas (por ejemplo, 1 USD = 0.85 EUR, 1 USD = 110 JPY, etc.).

Mostrar el resultado de la conversión.

Ejercicio 3: Generador de Contraseñas
Crea un programa que genere contraseñas aleatorias. La contraseña debe:

Tener una longitud mínima de 8 caracteres.

Incluir mayúsculas, minúsculas, números y caracteres especiales.

Mostrar la contraseña generada.

Ejercicio 4: Validación de Correo Electrónico
Escribe un programa que valide si una dirección de correo electrónico es válida. Para ello:

Pide al usuario que ingrese un correo electrónico.

Verifica que el correo tenga un formato válido (por ejemplo, que contenga un "@" y un dominio válido como ".com", ".es", etc.).

Muestra un mensaje indicando si el correo es válido o no.

Ejercicio 5: Juego de Adivinar el Número
Crea un juego en el que el programa elija un número aleatorio entre 1 y 100, y el usuario tenga que adivinarlo. El programa debe:

Indicar si el número ingresado por el usuario es mayor o menor que el número secreto.

Contar el número de intentos que lleva el usuario.

Mostrar un mensaje de felicitaciones cuando el usuario adivine el número.

Ejercicio 6: Manipulación de Strings
Escribe un programa que realice las siguientes operaciones sobre un string:

Pide al usuario que ingrese una frase.

Cuenta el número de palabras en la frase.

Cuenta el número de vocales y consonantes.

Invierte la frase y la muestra.

Convierte la frase a mayúsculas y luego a minúsculas.

Ejercicio 8: FizzBuzz
Escribe un programa que imprima los números del 1 al 100, pero:

Para múltiplos de 3, imprime "Fizz" en lugar del número.

Para múltiplos de 5, imprime "Buzz" en lugar del número.

Para múltiplos de ambos 3 y 5, imprime "FizzBuzz".

Ejercicio 9: Contador de Palabras Únicas
Crea un programa que cuente cuántas palabras únicas hay en un texto. El programa debe:

Pide al usuario que ingrese un texto.

Divide el texto en palabras.

Cuenta cuántas palabras únicas hay (ignorando mayúsculas y minúsculas).

Muestra el resultado.

Ejercicio 10: Simulador de Dados
Crea un programa que simule el lanzamiento de dos dados. El programa debe:

Generar dos números aleatorios entre 1 y 6.

Sumar los valores de los dados.

Mostrar un mensaje indicando si el resultado es "Bajo" (suma menor a 7), "Alto" (suma mayor a 7) o "7" (suma igual a 7).

Ejercicio 11: Ordenar un Array
Escribe un programa que ordene un array de números de menor a mayor sin usar el método sort(). El programa debe:

Pedir al usuario que ingrese una serie de números separados por comas.

Convertir la entrada en un array de números.

Ordenar el array manualmente (por ejemplo, usando el algoritmo de burbuja).

Mostrar el array ordenado.

Ejercicio 12: Detector de Palíndromos
Crea un programa que determine si una palabra o frase es un palíndromo (se lee igual de izquierda a derecha que de derecha a izquierda). El programa debe:

Pedir al usuario que ingrese una palabra o frase.

Ignorar espacios y mayúsculas/minúsculas.

Mostrar un mensaje indicando si es un palíndromo o no.
